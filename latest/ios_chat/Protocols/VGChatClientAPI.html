/**
 This method is used to create a new conversation with the provided name and display name.
 
 - Parameters:
    - name: The unique identifier for the conversation. This can be any alphanumeric string.
    - displayName: The display name for the conversation. This will be the name displayed to users in the UI.
    - callback: A callback function that will be called with the result of the conversation creation.
 It takes in two parameters:
 [Error] An exception object, if an error occurred during the creation.
 [ConversationId] The ID of the newly created conversation, if the creation was successful.
*/
- (void) createConversation:(NSString * _Nullable)name
                displayName:(NSString * _Nullable)displayName
                   callback:(nonnull void (^)(NSError * _Nullable error, VGConversationId _Nullable conversationId))callback

/**
 This method is used to retrieve a list of conversations that the user is a member of.
 
 - Parameters:
    - order: Optional parameter used to specify the order in which the conversations should be presented, default is ASC.
    - pageSize: Optional parameter to specify the number of conversations to be retrieved per page, default is 10.
    - cursor: Optional parameter to specify the starting point for pagination, default is the first page.
    - callback: A callback function to be called upon completion of the method.
 It takes in two parameters:
 [Error] An exception object, if an error occurred during the fetch.
 [VGConversationsPage] It will contain the list of conversations and the cursors for the next and previous page of results, if the fetch was successful.
*/
